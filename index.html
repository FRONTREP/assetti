Discovery de Rede

Como funciona

Este sistema roda inteiramente no navegador e, por restrições de segurança do próprio navegador, não consegue varrer a rede local automaticamente — nenhuma página web consegue fazer isso.

Em vez disso, você pode usar uma ferramenta externa de scan de rede como o Angry IP Scanner ou o Advanced IP Scanner (ambos gratuitos), exportar os resultados em .csv, e importar o arquivo aqui. O sistema reconhece automaticamente os dois formatos: colunas IP, Ping, Hostname, Ports (Angry IP Scanner) ou Status, Name, IP, Manufacturer, MAC Address (Advanced IP Scanner).

Depois de importado, você pode revisar cada dispositivo encontrado e decidir se ele já é um ativo cadastrado (vincular) ou se deve virar um novo ativo no inventário.
